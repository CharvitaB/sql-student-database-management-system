# Student Database Management System

A SQL project developed using MySQL for managing student records, courses, and marks.

## Features
- Store student details
- Manage course information
- Record student marks
- Perform JOIN operations
- Generate reports using SQL queries

## Technologies Used
- MySQL
- SQL

## Concepts Used
- CREATE DATABASE
- CREATE TABLE
- INSERT
- SELECT
- JOIN
- GROUP BY
- FOREIGN KEY
